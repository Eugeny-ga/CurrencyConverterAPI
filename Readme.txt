Конвертер валют.

Простой API-проект конвертера валют на Django REST.
Использует API сервера ЦБ РФ для получения актуального курса валют.

Требования:
Python - 3.11
Django - 4.2.5
Django REST Framework - 3.14.0


Установка:

Скачайте проект с Гитхаб и из корневой директории
выполните следующие команды:

pip install -r requirements.txt
python manage.py runserver


Запуск в Docker:
Скачайте проект с Гитхаб и из корневой директории
выполните следующие команды:

docker-compose up

Настройка:
Актуальный url и время хранения данных в кэше о курсе валют находятся в файле settings.py:
CASHED -> TIMEOUT (в сек, по умолчанию 10 мин)
RATES_URL
